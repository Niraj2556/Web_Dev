In Bootstrap, columns are a fundamental part of the grid system and allow you to create responsive layouts. Columns are used to divide the horizontal space within a row and organize content into a grid-based structure. Here's an overview of how to use columns in Bootstrap: Container: Wrap your rows and columns in a container to provide a fixed-width layout. Use the class .container for a fixed-width container or .container-fluid for a full-width container. Row: Inside the container, create a row using the class .row. Rows act as a horizontal grouping for columns and ensure proper alignment. Column Classes: Bootstrap provides a range of column classes that define the width and behavior of columns. The most commonly used classes are: .col: This class creates a column that automatically adjusts its width based on the available space. By default, it will occupy 100% of the row width on mobile devices and scale down on larger screens. .col-{breakpoint}-{columns}: Use these classes to create columns of specific widths at different breakpoints. For example, .col-sm-6 will create a column that occupies 50% of the row width on small screens and above. .col-{breakpoint}-auto: Use this class to create a column that automatically adjusts its width based on the content. For example, .col-lg-auto will create a column that takes up only the necessary width on large screens. Example In this example, we have created a container with a row containing two columns.
