Utilities In Bootstrap, utility classes are a set of predefined classes that you can apply directly to HTML elements to add various styling and functionality quickly. These utility classes are designed to handle common tasks like spacing, alignment, visibility, and grid layout. Here's an overview of some commonly used utility classes in Bootstrap: Alignment Utilities: Bootstrap includes classes for aligning elements: .text-{alignment}: Sets the text alignment to left, center, right, or justify. .align-{alignment}: Aligns elements vertically within a container. Replace {alignment} with top, middle, bottom, baseline, or stretch. Example
